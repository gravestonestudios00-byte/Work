# Orbit Foundry: Phone-only cloud APK build

This repo builds a debug APK through GitHub Actions.

## Build on phone

1. Open this repository on GitHub.
2. Go to Actions.
3. Select Build Android APK.
4. Tap Run workflow.
5. Open the completed run.
6. Download the OrbitFoundry-debug-apk artifact.
7. Unzip it.
8. Install app-debug.apk.

## Android install warning

If Android blocks the APK, open Settings, search Install unknown apps, allow your browser or file manager, then tap the APK again.

## Notes

This is a debug build, so Android may warn before installing. That is normal for apps built outside the Play Store.
